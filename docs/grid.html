@mixin grid-row-auto()

Output a css grid row with auto fit sizing and gutters

Parameters

$gutter: $grid-gutter (Number|Map)

The amount of gutter as number, breakpoint map (xs: 30px, md: 60px) or sizing map (min: 30px, max: 60px)

$min-size: (Number)

Minimum size of the grid columns

Example

scss 
.grid-row-auto {
  @include grid-row-auto(320px);
}
css compiled 
.grid-row-auto {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(min(100%, var(--min, 320px)), 1fr));
  gap: 1.875rem;
}

  grid-template-columns: repeat(var(--columns, 12), 1fr);
